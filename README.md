# LookaWin 🎲
https://looka.win

**Decentralized lottery on BNB Smart Chain (BSC)**

LookaWin is a trustless, immutable USDT lottery smart contract deployed on BNB Smart Chain. No admin keys. No upgrades. No manipulation — just code.

---

## How it works

- Buy a ticket for **2 USDT**
- Draws are triggered automatically via **Chainlink Automation**
- Winners are selected using **Chainlink VRF** (verifiable random function)
- Everything is on-chain and auditable on BscScan

## Prize distribution

| Pool | Share |
|------|-------|
| Hourly draw | 88% |
| Daily draw | 3% |
| Weekly draw | 2% |
| Monthly draw | 1% |
| Referral rewards | 3% |
| Protocol | 3% |

## Key properties

- **Immutable** — no owner can change parameters after deployment
- **Trustless** — randomness provided by Chainlink VRF
- **Transparent** — fully open-source under BUSL-1.1

## Network

- **Chain**: BNB Smart Chain (BSC) Mainnet
- **Token**: USDT (BEP-20)
- **Min participants**: 50 per draw
